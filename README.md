# Ionic 3 API CRUD
Exemplo de como fazer requisições HTTP a uma API REST e também como paginar os resultados utilizando o [InfiniteScroll](https://ionicframework.com/docs/api/components/infinite-scroll/InfiniteScroll/).

## Api Fake
